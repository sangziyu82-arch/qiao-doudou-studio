# 敲豆豆工作台功能说明

## 项目定位

敲豆豆工作台是一个把图片转换成像素格子底板的网页工具，适合用于敲豆豆、拼豆、像素底板、3D 打印方块排布等场景。用户可以上传图片，裁剪需要生成的区域，把图片转换成一个个可编辑的小方格，再根据 STL 模型尺寸导出图纸或排布后的 STL 文件。

## 核心流程

1. 上传一张图片。
2. 在裁剪窗口中框选需要生成底板的部分。
3. 设置横向格数、颜色数量、背景阈值和尺寸参数。
4. 系统根据图片生成像素色块网格。
5. 用户可以用画笔、橡皮、吸管继续手动修色块。
6. 打开原图叠加层，对照图片轮廓微调。
7. 选择 STL 模型尺寸，生成真实尺寸排布。
8. 导出 PNG 图纸或导出已经排好位置的 STL 模型。

## 图片上传与裁剪

- 支持点击上传图片。
- 上传后不会直接生成，而是先进入裁剪窗口。
- 裁剪框可以拖动位置，也可以拖动边缘和角点调整大小。
- 确认裁剪后，只会根据选中的区域生成像素底板。
- 这样可以避免图片边缘、空白区域或不需要的部分影响生成结果。

## 像素网格生成

- 根据横向格数自动计算纵向格数，保持裁剪区域比例。
- 根据颜色数量参数提取主要颜色。
- 支持自动清除浅色背景。
- 支持背景阈值调节，便于处理透明背景、白底图、浅色底图。
- 生成结果会显示：
  - 网格数量。
  - 有色方格数量。
  - 实际成品尺寸。
  - 单格 STL 尺寸。

## 网格编辑工具

### 画笔

- 使用当前颜色给格子上色。
- 可以点击单个格子修改颜色。
- 按住鼠标拖过格子时，可以连续绘制。

### 橡皮

- 把格子删除为空格。
- 空格不会参与颜色统计，也不会导出为 STL 方块。

### 吸管

- 点击已有颜色格子，可以吸取该格子的颜色。
- 点击空白格子，会吸取真正的白色 `#FFFFFF`。
- 吸取后会自动切回画笔，方便马上继续绘制。

## 当前颜色

- 工具栏右侧显示当前画笔颜色。
- 支持用颜色选择器手动修改颜色。
- 颜色以十六进制形式显示，例如 `#C0C0C0`。

## 原图叠加层

- 点击“原图”按钮，可以在网格上方显示原始图片。
- 支持透明度滑块调节原图透明度。
- 原图叠加层和像素网格使用同一套裁剪坐标，方便对齐轮廓。
- 用户可以一边看原图，一边修改色块。
- 快捷键：
  - `Ctrl + 1`：显示或隐藏原图叠加层。

## RGB 颜色表

- 右侧会自动统计当前网格中所有颜色。
- 每个颜色会显示：
  - 编号。
  - 色块预览。
  - RGB 十六进制颜色值。
  - 该颜色出现的格子数量。
- 点击颜色行，会把该颜色设为当前画笔颜色。
- 同时，画布中所有相同颜色的格子会发光，方便快速定位。
- 修改某个发光格子的颜色后，只有被改掉的那个格子不再发光，其他同色格子会继续保持高亮。
- 快捷键：
  - `Esc`：取消颜色定位发光。

## STL 模型功能

### 模型来源

项目默认扫描本地路径：

```text
D:\下载\3dmox
```

运行：

```bash
npm run scan-models
```

脚本会扫描 STL 文件，读取模型尺寸，并复制到：

```text
public/models/
```

同时生成：

```text
public/stl-models.json
```

### 模型选择

- 左侧“3D STL 模型”区域可以选择扫描到的 STL 模型。
- 选择模型后，会显示模型的 X、Y、Z 尺寸。
- 点击“应用 STL 尺寸并设为无间距”后：
  - 单格宽度会使用 STL 模型 X 尺寸。
  - 模型间距会设为 `0`。
  - 成品尺寸会按模型真实尺寸重新计算。

### 模型链接

- 支持下载原始 STL 模型。
- 支持复制 STL 模型引用链接。
- 支持复制模型清单链接。
- 局域网用户也可以通过链接下载或引用模型。

## 导出功能

### PNG 图纸

- 点击“PNG 图纸”或“导出图纸”可以导出当前网格图纸。
- 图纸会包含当前格子颜色和网格线。
- 适合用于打印、对照、人工拼色。

### 排布后的 STL 模型

- 点击“STL 模型”可以导出已经按网格排好的 STL 文件。
- 每一个有颜色的格子都会对应一个 STL 方块。
- 空格不会生成 STL 方块。
- 排布规则：
  - 横向间距 = 单格 STL 宽度 + 模型间距。
  - 纵向间距 = 单格 STL 高度 + 模型间距。
  - 默认可以实现模型之间无间距排列。

## 示例图

- 项目内置一张示例图。
- 点击“示例”可以恢复默认示例、默认参数和默认识别设置。
- 当前默认参数包括：
  - 横向格数：`18`
  - 颜色数量：`18`
  - 背景阈值：`211`
  - 单格宽度：`1 mm`
  - 模型间距：`0 mm`

## 本地草稿

- 点击“本地草稿”会把当前网格和参数保存到浏览器本地存储。
- 当前版本主要提供保存入口，后续可以继续扩展为草稿列表、恢复草稿、多版本记录等功能。

## 注意事项

- 白色格子和空格不是同一个概念。
- 白色格子是 `#FFFFFF`，会参与统计和 STL 导出。
- 空格是没有颜色的格子，不参与统计，也不会导出 STL 方块。
- 如果想删除一个格子，请使用橡皮。
- 如果想画白色方块，可以用吸管点击空白格，或在颜色选择器里选择白色。
